# Twitter Sentiment Analysis

Classifies tweets as **Positive** or **Negative** using NLP and Machine Learning.

## Dataset
Sentiment140 — 1.6M labeled tweets from [Kaggle](https://www.kaggle.com/datasets/kazanova/sentiment140)

> Dataset is not included in this repo due to size. Download it from Kaggle and place `sentiment140.zip` in the project root.

## Tech Stack
- Python, NLTK, Scikit-learn
- TF-IDF Vectorization
- Logistic Regression

## Results
| Data | Accuracy |
|------|----------|
| Training | 79.8% |
| Testing | 77.6% |

## How to Run
1. Download dataset from Kaggle
2. Place `sentiment140.zip` in project root
3. Run all cells in `Twitter_Sentiment_Analysis.ipynb`
